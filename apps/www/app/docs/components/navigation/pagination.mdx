# Pagination

Components and hooks for navigating through paged data with cursor-based and offset-based pagination strategies.

## Cursor Pagination

A pagination component for use with cursor-based pagination. Cursor-based pagination loads data in chunks by using a cursor from the last item on the current page to know where to start the next set. It doesn't let you jump to a specific page or know how many total pages there are, but it's more efficient for large or real-time data sets.

<Example className="flex flex-col gap-6">
	<CursorPagination.Root defaultPageSize={100}>
		<CursorPagination.PageSizeSelect />
		<CursorPagination.Buttons hasNextPage hasPreviousPage />
	</CursorPagination.Root>
	<CursorPagination.Root defaultPageSize={100}>
		<CursorPagination.PageSizeValue />
		<CursorPagination.Buttons hasNextPage hasPreviousPage />
	</CursorPagination.Root>
	<CursorPagination.Root defaultPageSize={100}>
		<CursorPagination.Buttons hasNextPage hasPreviousPage={false} />
	</CursorPagination.Root>
</Example>

## Within a data table

`CursorPagination` pairs with a [`DataTable`](/components/data-display/data-table) — drive its `Buttons` and `PageSizeSelect` from a TanStack Table instance.

**Client-paginated** — the table holds every row and `getPaginationRowModel()` slices them. Read `getCanPreviousPage()` / `getCanNextPage()` for the buttons and call `setPageSize()` / `setPageIndex(0)` when the size changes:

<Example>
	<PaginatedPaymentsDemo />
</Example>

```tsx
import {
	DataTable,
	createColumnHelper,
	getCoreRowModel,
	getPaginationRowModel,
	useReactTable,
} from "@ngrok/mantle/data-table";
import { CursorPagination } from "@ngrok/mantle/pagination";

// defaultPageSize seeds an UNCONTROLLED <Select>, so keep it stable — a module
// const (or the table's INITIAL page size), never the live page size.
const DEFAULT_PAGE_SIZE = 10;

type Payment = {
	id: string;
	amount: number;
	status: "pending" | "processing" | "success" | "failed";
	email: string;
};

const columnHelper = createColumnHelper<Payment>();

const columns = [
	columnHelper.accessor("email", {
		id: "email",
		header: () => <DataTable.Header>Email</DataTable.Header>,
		cell: (props) => <DataTable.Cell>{props.getValue()}</DataTable.Cell>,
	}),
	columnHelper.accessor("status", {
		id: "status",
		header: () => <DataTable.Header>Status</DataTable.Header>,
		cell: (props) => <DataTable.Cell>{props.getValue()}</DataTable.Cell>,
	}),
	columnHelper.accessor("amount", {
		id: "amount",
		header: () => <DataTable.Header className="text-right">Amount</DataTable.Header>,
		cell: (props) => <DataTable.Cell className="text-right">${props.getValue()}</DataTable.Cell>,
	}),
];

function PaginatedTable({ data }: { data: Payment[] }) {
	const table = useReactTable({
		data,
		columns,
		getCoreRowModel: getCoreRowModel(),
		getPaginationRowModel: getPaginationRowModel(),
		initialState: { pagination: { pageSize: DEFAULT_PAGE_SIZE } },
	});

	const rows = table.getRowModel().rows;

	return (
		<div className="flex flex-col gap-4">
			<DataTable.Root table={table}>
				<DataTable.Head />
				<DataTable.Body>
					{rows.length > 0 ? (
						rows.map((row) => <DataTable.Row key={row.id} row={row} />)
					) : (
						<DataTable.EmptyRow>No results.</DataTable.EmptyRow>
					)}
				</DataTable.Body>
			</DataTable.Root>
			<CursorPagination.Root className="flex justify-end" defaultPageSize={DEFAULT_PAGE_SIZE}>
				<CursorPagination.PageSizeSelect
					onChangePageSize={(size) => {
						table.setPageSize(size);
						table.setPageIndex(0); // reset to the first page when the size changes
					}}
				/>
				<CursorPagination.Buttons
					hasPreviousPage={table.getCanPreviousPage()}
					hasNextPage={table.getCanNextPage()}
					onPreviousPage={() => table.previousPage()}
					onNextPage={() => table.nextPage()}
				/>
			</CursorPagination.Root>
		</div>
	);
}
```

**Server / cursor data** — the server returns one page at a time, so there is no `getPaginationRowModel` and no total count. Wire `Buttons` to your fetch's has-next / has-prev and load callbacks, and show the fixed, server-controlled page size with the read-only `PageSizeValue` instead of `PageSizeSelect`. The example simulates the server's cursor state with a page index:

<Example>
	<ServerCursorPaginationExample />
</Example>

```tsx
import { CursorPagination } from "@ngrok/mantle/pagination";
import { useState } from "react";

// The page size is fixed by the server, so surface it read-only with PageSizeValue.
const PAGE_SIZE = 10;
const TOTAL_PAGES = 3;

function ServerCursorPagination() {
	// In a real app the has-prev / has-next flags come from your fetch's cursors;
	// here we simulate them with a page index.
	const [pageIndex, setPageIndex] = useState(0);

	return (
		<CursorPagination.Root className="flex justify-end" defaultPageSize={PAGE_SIZE}>
			<CursorPagination.PageSizeValue />
			<CursorPagination.Buttons
				hasPreviousPage={pageIndex > 0}
				hasNextPage={pageIndex < TOTAL_PAGES - 1}
				onPreviousPage={() => setPageIndex((index) => Math.max(0, index - 1))}
				onNextPage={() => setPageIndex((index) => Math.min(TOTAL_PAGES - 1, index + 1))}
			/>
		</CursorPagination.Root>
	);
}
```

See the [DataTable pagination recipe](/components/data-display/data-table#pagination-controls) for the full table markup.

## Composition

Compose the parts of a `CursorPagination` together to build your own:

```text showLineNumbers=false
CursorPagination.Root
├── CursorPagination.PageSizeSelect
├── CursorPagination.PageSizeValue
└── CursorPagination.Buttons
```

## Polymorphism

`CursorPagination.PageSizeValue` supports the `asChild` prop so you can render the read-only page size onto your own element instead of the default `span`.

```tsx
import { CursorPagination } from "@ngrok/mantle/pagination";

<CursorPagination.Root defaultPageSize={100}>
	<CursorPagination.PageSizeValue asChild>
		<strong className="text-strong" />
	</CursorPagination.PageSizeValue>
	<CursorPagination.Buttons hasNextPage hasPreviousPage />
</CursorPagination.Root>;
```

## API Reference

### CursorPagination.Root

The root container for cursor-based pagination. Manages the page size state.

All props from the HTML `div` element, plus:

| Prop              | Type     | Default | Description                           |
| :---------------- | :------- | :------ | :------------------------------------ |
| `defaultPageSize` | `number` |         | The default number of items per page. |

### CursorPagination.Buttons

A pair of previous/next buttons for navigating between pages.

| Prop              | Type         | Default | Description                                               |
| :---------------- | :----------- | :------ | :-------------------------------------------------------- |
| `hasNextPage`     | `boolean`    |         | Whether there is a next page of data to load.             |
| `hasPreviousPage` | `boolean`    |         | Whether there is a previous page of data to load.         |
| `onNextPage?`     | `() => void` |         | Callback called when the next page button is clicked.     |
| `onPreviousPage?` | `() => void` |         | Callback called when the previous page button is clicked. |

### CursorPagination.PageSizeSelect

A select input for changing the number of items per page.

| Prop                | Type                      | Default                | Description                                    |
| :------------------ | :------------------------ | :--------------------- | :--------------------------------------------- |
| `pageSizes?`        | `readonly number[]`       | `[5, 10, 20, 50, 100]` | A list of page sizes to choose from.           |
| `onChangePageSize?` | `(value: number) => void` |                        | Callback called when the page size is changed. |

### CursorPagination.PageSizeValue

Displays the current page size as a read-only value (e.g. "100 per page").

| Prop       | Type      | Default | Description                                                           |
| :--------- | :-------- | :------ | :-------------------------------------------------------------------- |
| `asChild?` | `boolean` | `false` | Compose onto an alternative element type or your own React component. |

## Offset Pagination

### useOffsetPagination

A headless hook for managing offset-based pagination state. Offset-based pagination lets you jump to a specific page and know the total number of pages.

<Example>
	<OffsetPaginationExample />
</Example>

```tsx
import { Button } from "@ngrok/mantle/button";
import { useOffsetPagination } from "@ngrok/mantle/pagination";

function OffsetPaginationControls() {
	const pagination = useOffsetPagination({ listSize: 150, pageSize: 10 });

	return (
		<div className="flex items-center gap-4">
			<p>
				Page {pagination.currentPage} of {pagination.totalPages}
			</p>
			<Button
				type="button"
				appearance="outlined"
				intent="neutral"
				disabled={!pagination.hasPreviousPage}
				onClick={pagination.previousPage}
			>
				Previous
			</Button>
			<Button
				type="button"
				appearance="outlined"
				intent="neutral"
				disabled={!pagination.hasNextPage}
				onClick={pagination.nextPage}
			>
				Next
			</Button>
		</div>
	);
}
```

| Parameter  | Type     | Description                            |
| :--------- | :------- | :------------------------------------- |
| `listSize` | `number` | The total number of items in the list. |
| `pageSize` | `number` | The number of items per page.          |

The hook returns an object with the current pagination state and navigation helpers:

| Property          | Type                     | Description                                                 |
| :---------------- | :----------------------- | :---------------------------------------------------------- |
| `currentPage`     | `number`                 | The current page number, 1-indexed (starting at 1).         |
| `totalPages`      | `number`                 | The total number of pages.                                  |
| `hasPreviousPage` | `boolean`                | Whether there is a previous page.                           |
| `hasNextPage`     | `boolean`                | Whether there is a next page.                               |
| `offset`          | `number`                 | The offset of the current page within the list.             |
| `pageSize`        | `number`                 | The current number of items per page.                       |
| `nextPage`        | `() => void`             | Go to the next page.                                        |
| `previousPage`    | `() => void`             | Go to the previous page.                                    |
| `goToPage`        | `(page: number) => void` | Go to a specific page (clamped to the valid range).         |
| `goToFirstPage`   | `() => void`             | Go to the first page.                                       |
| `goToLastPage`    | `() => void`             | Go to the last page.                                        |
| `setPageSize`     | `(size: number) => void` | Set the number of items per page; resets to the first page. |

### getOffsetPaginatedSlice

A utility function to get a paginated slice of a list based on the current offset pagination state. Pair it with `useOffsetPagination` to render just the current page of items.

<Example>
	<OffsetPaginatedSliceExample />
</Example>

```tsx
import { Button } from "@ngrok/mantle/button";
import { getOffsetPaginatedSlice, useOffsetPagination } from "@ngrok/mantle/pagination";

const items = ["Alpha", "Bravo", "Charlie", "Delta", "Echo", "Foxtrot", "Golf", "Hotel", "India"];

function PaginatedList() {
	const pagination = useOffsetPagination({ listSize: items.length, pageSize: 3 });
	// Slice the current page out of the full list: ["Alpha", "Bravo", "Charlie"] on page 1,
	// ["Delta", "Echo", "Foxtrot"] on page 2, and so on.
	const currentPageItems = getOffsetPaginatedSlice(items, pagination);

	return (
		<div className="flex flex-col gap-4">
			<ul>
				{currentPageItems.map((item) => (
					<li key={item}>{item}</li>
				))}
			</ul>
			<div className="flex items-center gap-4">
				<p>
					Page {pagination.currentPage} of {pagination.totalPages}
				</p>
				<Button
					type="button"
					appearance="outlined"
					intent="neutral"
					disabled={!pagination.hasPreviousPage}
					onClick={pagination.previousPage}
				>
					Previous
				</Button>
				<Button
					type="button"
					appearance="outlined"
					intent="neutral"
					disabled={!pagination.hasNextPage}
					onClick={pagination.nextPage}
				>
					Next
				</Button>
			</div>
		</div>
	);
}
```
